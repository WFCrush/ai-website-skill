# website skill 使用说明

本目录是本地 AI 建站工作流 skill 包。

## 调用建议

当用户需要创建网站、优化网站颜值、制定 UI 规范、实现 Landing Page、审查截图或部署上线时，可以使用 `website` skill。

## 目标

帮助小白用 AI 按专业流程完成网站项目：

1. 需求分析
2. 视觉风格定位
3. UI 设计规范
4. 首页结构与文案
5. 前端实现
6. 截图审查与迭代
7. 部署上线

## 工作原则

- 不要一上来就写代码，先确认需求、风格、规范。
- 不让 AI 自由发挥，要用参考网站、设计规范和模块化流程约束结果。
- 优先推荐成熟技术栈：Next.js、TypeScript、Tailwind CSS、shadcn/ui、lucide-react。
- 优先选择简单稳定的部署方式：Vercel、Netlify、Cloudflare Pages。
- 对小白解释时要一步一步讲，不默认用户懂 Git、命令行、服务器或前端工程化。
